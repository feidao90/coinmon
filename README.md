# coinmon
利用命令行，实时产看比特币价格变化：
  确保你的系统上安装了 Node.js 和 Npm，从终端运行以下命令安装 Coinmon。
  
  sudo npm install -g coinmon
  运行coinmon,查看前10大比特币价格:
  conimon是从 coinmarketcap.com API 获取所有详细信息。
